# Signet App (Kotlin Multiplatform)

Consumer example for the `signet` Kotlin Multiplatform library. This directory keeps
the framework aligned with the `signet` plus `SignetApp` layout shared by the other
bindings.

The example is generated from the Kotlin Multiplatform wizard (kmp.new) once the
binding surface is defined, rather than hand-written; it stays on the official
project scaffold.
